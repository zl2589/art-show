# art-show v1.0.0

## Installation

Using :

```js
$ const artShow = require("art-show");
```

## Examples

```javascript
const artShow = require("art-show");

artShow.createArts("nanchen");
artShow.createArts("nanchen", { fontFamily: "crazy" });
```
